NHL Semester Project

Authors: Alyssa Kutney, Dylan Shapiro

Our project is an NHL Hockey based project. We display information for NHL Teams, NHL Players, and Upcoming NHL Games.

index.html
Main page of our project. Displays Team stats for a selected NHL Team.
Hockey.js
Javascript file that contains multiple API calls for Team Name, Stats, Rosters, and Team Logo.
NHLGames.html
HTML file that displays any upcoming games for the selected Team.
NHLGames.js
Javascript file that contains API calls for Team's upcoming regular season game schedule and logo.
NHLPeople.html
HTML file that displays an NHL Player, their stats, team logo and an image of themselves.
NHLpeople.js
Javascript file that contains API calls for a Team's specific player, their stats, team logo, and an image of the player.
main.css
CSS file that contains the styles and classes that are used throughout all html pages in this project.
app.js
Javascript file that takes care of the project's header, footer, and theme.
